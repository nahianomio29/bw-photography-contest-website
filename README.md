# Black & White Photography Contest Website

## Description
A multi-page website built for a black-and-white photography contest, allowing participants to view contest rules, browse past winners, vote for their favorite photographs, and submit their own entries.

## Pages
- **Home** (index.html) — Welcome page with auto-rotating image slideshow and live chatroom
- **Rules & Regulations** (rules.html) — Contest rules, categories, and deadlines
- **2023 Winners** (winners.html) — Gallery showcasing winning photos from each category
- **Voting** (voting.html) — Voting form with photo display for each category finalist
- **Submit Photography** (submit.html) — Photo submission form with file upload
- **Contact Us** (contact.html) — Contact form and embedded live chatroom

## Features
- Auto-rotating image slideshow using jQuery and JavaScript
- Photo voting system with radio button selection per category (Vehicle, Bridges & Waterfalls, Buildings)
- Photo submission form with file upload support (JPG, JPEG, PNG, GIF, RAW)
- Contact form with name, email, and message fields
- Embedded live chatroom on Home and Contact pages
- Custom font (News Cycle) and consistent CSS styling across all pages
- SEO optimized with meta descriptions, keywords, and robots directives
- XML sitemap for search engine indexing
- Responsive viewport for mobile compatibility
- Dynamic footer navigation generated via JavaScript

## Technologies Used
- HTML5
- CSS3
- JavaScript
- jQuery
- Modernizr (HTML5 feature detection)

## How to Run
1. Clone or download the repository
2. Open `index.html` in any web browser
3. Navigate between pages using the top navigation bar or footer links
